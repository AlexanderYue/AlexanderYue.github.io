About Me

I’m a Computer Science student with a strong focus on cybersecurity, artificial intelligence, and high-performance computing. My work spans from developing secure software systems to exploring how CUDA and GPU parallelization can accelerate real-world applications like rendering and n-body simulations. I’ve built projects ranging from a Flask-based admissions system with SQL Server on Azure, to a CUDA-powered circle renderer and particle simulator, to NLP-driven sentiment analysis tools for large-scale datasets. These experiences have sharpened my skills in Python, C++, SQL/NoSQL, and cloud technologies (AWS, Azure), as well as frameworks for machine learning, NLP, and secure system design. I’m currently pursuing opportunities to apply my expertise in cybersecurity and AI—particularly in areas like threat detection, OSINT workflows, and AI-driven analytics—while continuing to explore the intersection of parallel computing and security challenges.
