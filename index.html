NLP Amazon Review Engine

Language: Python

This project is an Natural Language Processing (NLP)–based search engine designed to extract opinions from Amazon product reviews. Instead of just keyword matching, it focuses on capturing the sentiment and context behind reviews to deliver more meaningful results.

Goal: Allow users to search reviews using natural queries like “audio quality: poor” or “gps map: useful” and retrieve the most relevant customer feedback.

Dataset: Worked with a preprocessed Amazon review dataset (reviews_segment.pkl).

Methods:

  • Baseline: A Boolean keyword matching model.
  • M1 (Improved Model): Enhanced approach with smarter filtering (e.g., sentiment detection).
  • M2 (Best Model): Incorporated advanced NLP techniques like sentiment analysis, dependency parsing, and semantic similarity scoring.

Evaluation: Models were evaluated on precision, measuring how accurately the system returned relevant reviews.

Tech Stack: Python, scikit-learn, NLTK/spaCy, and custom NLP pipelines.

CUDA Linear Algebra Solver

Language: C++ / CUDA

This project implements a dense LU decomposition solver in C++/CUDA to accelerate solving systems of linear equations (Ax = b). GPU optimization techniques were explored to improve performance over CPU implementations.

Goal: Implement a parallel LU decomposition solver to efficiently solve dense systems of linear equations on the GPU.

Dataset: Used generated test matrices (3x3, 4x4, 5x5, etc.) to validate correctness and benchmark performance.

Methods:

  • Stage 1 (CPU Baseline): Implemented serial LU decomposition with forward/backward substitution in C++.
  • Stage 2 (CUDA Parallelization): Ported LU decomposition to CUDA with row operations and GPU kernels for substitution.
  • Stage 3 (Optimization): Applied tiling, shared memory, coalesced memory access, and thread/block decomposition to maximize GPU efficiency.

Evaluation: Benchmarked runtime and scalability, demonstrating significant performance improvements over CPU implementations for larger matrices.

Tech Stack: C++, CUDA, NVIDIA GPU architecture, benchmarking tools.
